Sobre

Sou desenvolvedor com 2 anos e 9 meses de experiência, com sólida atuação no desenvolvimento de soluções utilizando Java, Kotlin e Node.js. Tenho experiência na criação de testes unitários, de regressão e End-to-End (com Cypress), garantindo a qualidade do código. Além disso, sou proficiente no uso de ferramentas de monitoramento como New Relic, e na análise de qualidade de código com SonarCloud e Fortify.

Tenho um bom domínio em bancos de dados relacionais (PostgreSQL) e não-relacionais (MongoDB), além de experiência com pipelines de CI/CD usando Jenkins e GitHub Actions. Também trabalhei com Lambdas serverless e Angular, ampliando minha versatilidade nas tecnologias utilizadas.

Busco oportunidades para contribuir com projetos desafiadores, aplicando boas práticas de desenvolvimento e colaborando em equipes multidisciplinares, sempre visando a entrega de soluções eficientes e de qualidade.

Habilidades Técnicas:

  • Linguagens: Java, Kotlin, Node.js, JavaScript
  • Bancos de Dados: PostgreSQL, MongoDB
  • Ferramentas: Jenkins, GitHub Actions, New Relic, SonarCloud, Fortify, Cypress
  • Metodologias: CI/CD, Testes Automatizados, APIs REST
  • Outros: Lambda (AWS), Angular, Git
